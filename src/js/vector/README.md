### new Vector([x], [y])

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| [x] | <code>number</code> | <code>0</code> | X value |
| [y] | <code>number</code> | <code>0</code> | Y value |

**Example**  
```js
const x = 1;
    const y = 1;
    const vector = new Vector(x, y);
```
